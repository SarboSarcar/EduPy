EduPy:
The Journey

The EduPy Project started as a school assignment implementing data mining skills using Python based frameworks for the creation of an undergraduate course finding and recommending system.
The Project relies heavily on the websites study.eu and unsplash.com for mining the course data and adding the background images respectively. The tools and frameworks implemented, have been listed here:
  • Python 3 (for back-end and server-side development)
  • HTML 5 (for front-end development and user interface construction)
  • CSS 3 (for styling static pages)
  • Jinja 2 (Python based templating engine for markup)
